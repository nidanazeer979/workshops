📘 Workshops Lab – Jupyter Notebook
📌 Overview

This lab contains practical exercises implemented in a Jupyter Notebook (workshops.ipynb). It is designed to help understand basic to intermediate concepts through hands-on coding.

🎯 Objectives
Practice core programming concepts
Understand problem-solving techniques
Work with data and logic in Python
Build small functional tasks step-by-step
🛠️ Requirements

Before running this notebook, make sure you have:

Python 3.x installed
Jupyter Notebook / JupyterLab
Required libraries (if any)

Install Jupyter if needed:

pip install notebook
▶️ How to Run
Open terminal / command prompt
Navigate to the project folder
Run:
jupyter notebook
Open workshops.ipynb
Run cells step by step
